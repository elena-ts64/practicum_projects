# Предсказание увольнения сотрудников из компании

## Описание проекта
Исследование необходимо для HR отдела компании «Работа с заботой» для предсказания оттока сотрудников и минимизации рисков компании, связанных с увольнениями.

Компания предоставила данные с характеристиками сотрудников компании. Среди них — уровень удовлетворённости сотрудника работой в компании. Эту информацию получили из форм обратной связи: сотрудники заполняют тест-опросник, и по его результатам рассчитывается доля их удовлетворённости от 0 до 1, где 0 — совершенно неудовлетворён, 1 — полностью удовлетворён. 
Собирать данные такими опросниками не так легко: компания большая, и всех сотрудников надо сначала оповестить об опросе, а затем проследить, что все его прошли. Поэтому данный показатель - уровень удовлетворённости сотрудника - необходимо предсказывать с помощью машинного обучения. Почему бизнесу это важно: удовлетворённость работой напрямую влияет на отток сотрудников. А предсказание оттока — одна из важнейших задач HR-аналитиков. Внезапные увольнения несут в себе риски для компании, особенно если уходит важный сотрудник. 
Вторая задача — построить модель, которая сможет на основе данных заказчика предсказать то, что сотрудник уволится из компании.

Цель исследования:
- Минимизировать финансовые потери компании, связанные с оттоком сотрудников.

Задачи исследования:
1. Построить модель машинного обучения, которая сможет на основе данных заказчика предсказать уровень удовлетворённости сотрудника, необходимого для предсказания оттока.
2. Построить модель машинного обучения, которая сможет на основе данных заказчика и пердсазанного уровня удовлетворённости сотрудника предсказать увольнение сотрудника из компании.

## Навыки и инструменты
python, pandas, matplotlib, seaborn, numpy, scipy, shap, phik, scikit-learn\
выбор модели МО, подбор гиперпараметров, использование пайплайнов, анализ признаков

## Общий вывод
В рамках проекта необходимо было выполнить 2 задачи:
1. построение модели, которая сможет предсказать уровень удовлетворённости сотрудника (задача регрессии)
2. построение модели, которая сможет предсказать увольнение сотрудника из компании (задача классификации)

В ходе работы над 1 задачей:
* была выполнена загрузка данных и их просмотр
* была выполнена предобработка данных:
    * выявлены пропуски в категорильных параметрах, обработка которых добавлена в пайплан
    * удалены дубликаты в тренировочных данных
    * выполнена проверка на наличие неявных дубликатов
    * выявлены опечатки в значениях категориального параметра level, обработка которых добавлена в пайплан
    * выявлены пропуски в значениях в виде пробелов, обработка которых добавлена в пайплан
* был выполнен исследовательский анализ данных:
    * ознакомление с распределениями числовых признаков
    * ознакомление с распределениями катеегориальных признаков
    * корреляционный анализ признаков
    * проверка мультиколлинеарности входных признаков
* была выполнена подготовка данных:
    * разбиение признаков в зависимости от их типа
    * добавление в пайплайн кодирования и масштабирования признаков в зависимости от их типа
* были обучены модели регресии с подбором гиперпараметров, оценка качества модели выполнялась на основе новой метрики SMAPE, расчёт которой так же был добавлен
* на подобранной и обученной моделе RandomForestRegressor было выполнено предсказание на тестовой выборке, предоставленной заказчиком, для проверки удовлетворения требованию по значению метрики SMAPE

В ходе работы над 2 задачей:
* была выполнена загрузка данных и их просмотр (список входнвх признаков остался тем же, что был в задаче 1)
* была выполнена предобработка данных в виде удаления дубликатов и использования пайплайнов из 1 задачи
* был выполнен исследовательский анализ данных:
    * рассмотрено распределение нового целевого признака
    * выполнена оценка корреляции с новых входным признаком
    * сосавлен портрет уволившегося сотрудника: Больший уровень увольнений у сотрудников с более низким уровнем зарплаты и с более низким уровнем удовлетворённости. При это отдел практически не влияет на количество увольнений
* моделью из задачи 1 были предсказаны уровни удовлетворения сотрудников и добавлены во входные признаки для модели задачи 2
* была выполнена подготовка данных:
    * целевой признак закодирован при помощи LabelEncoder
    * собран общий пайплан, включающий предобработку входных признаков с учётом их типа, включающий новый праметр удовлетворения сотрудников для масштабирования
* были обучены модели классификации с подбором гиперпараметров, оценка качества модели выполнялась на основе метрики ROC-AUC
* был проведён анализ влияния признаков на предсказание
* на подобранной и обученной моделе RandomForestClassifier с подобранными параметрами 'min_samples_split'=10, 'max_depth'=5, 'criterion'='log_loss' было выполнено предсказание на тестовой выборке, предоставленной заказчиком

Подтвердилась зависимость удовлетворённости сотрудника с возможностью уволнения. Чем ниже стаж работы в компании, зарплата, уровень должности, удовлетворённость, оценка руководителя, а так же чем выше загруженность сотрудника, тем больше вероятность увольнения. Данная картина выглядит вполне разумно. Созданная модель будет полезна в отслеживании аномалий в данной зависимости. При постоянном выполнении предказаний для сотрудников можно выявлть возможность увольнения ценных сотрудиков (работающих долго, либо имеющих высокий уровень), чтобы заранее принимать меры для повышения удовлетворённости сотрудника и сохранении его в штате.
